# Context Search Assistant

A Chrome extension that enables inline search and note-taking without breaking workflow.

## Problem
While studying or researching, users constantly switch tabs to search small queries, which breaks focus and creates clutter.

## Solution
This tool allows users to:
- Open a search panel instantly using a keyboard shortcut
- Search without leaving the page
- Take and save notes alongside their work

## Features
- Floating draggable search panel
- Keyboard shortcut (Ctrl + Shift + Space)
- Inline search (prototype)
- Notes saving system
